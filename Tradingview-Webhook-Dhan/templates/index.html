How to Set Up in TradingView

  1. Open your TradingView chart
  2. Click on "Alerts" (bell icon)
  3. Click "Create Alert"
  4. Set your alert conditions
  5. Under "Notifications", click "Webhook URL"
  6. Enter your webhook URL: http://your-server-ip/webhook
  7. In the "Message" field, copy and paste the JSON message from below
  8. Click "Create" to save the alert
Important: Make sure to copy the exact JSON format for each signal. The webhook server expects the message in this specific format.
